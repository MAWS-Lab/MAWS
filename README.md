## Overview

MAWS est un système d'exploitation souverain pour agents IA multimodaux, conçu pour :
- **Souveraineté totale** : données et exécution sous contrôle exclusif
- **Sécurité maximale** : confinement GHOST-ALPHA, obfuscation ε, zero-knowledge split inference
- **Performance** : cache sémantique natif, exécution sandboxée sécurisée
- **Consensus utile** : Proof of Intelligence (PoI) au lieu de PoW inutile

Projet NANASHI — JANUS phase active (2026).

## Key Components

- **WAKA-ENGINE** : cœur exécutable privé (main.py)
- **GHOST-ALPHA VAULT** : confinement total des données sensibles
- **TUNNEL USL-EDL** : transport sécurisé AES-GCM + ε-noise
- **LAYER MUMEI** : couche publique avec split inference & zero-knowledge
- **AGENTS GHOSTS** : Blinky (sync), Inky (bruit dynamique), Pinky (purge)
- **Proof of Intelligence** : consensus basé sur travail IA utile
